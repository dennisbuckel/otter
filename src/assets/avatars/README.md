## How to Use

1. Place your avatar images in this folder with the naming convention `avatar{userId}.jpg` (e.g., `avatar1.jpg`, `avatar2.jpg`, etc.)
2. The system will automatically use these images for the corresponding users
3. If an avatar image is not found, the system will fall back to using the generated avatar with the user's initial

## Recommended Image Specifications

- **Format**: JPG or PNG
- **Size**: 500x500 pixels (square)
- **File size**: Keep under 200KB for optimal performance
